BitGPT as a finite state machine

A tiny causal autoregressive transformer for next-bit prediction.

This demo models a causal autoregressive generator of binary strings over the alphabet {0,1}. For a fixed context length k, it learns the conditional distribution of the next bit from the previous k bits, using one training family at a time, such as copy, invert, XOR, majority, or an empirical sequence. Since there are only 2k possible contexts, the problem can be viewed as a finite-state machine whose states are the length-k contexts and whose transitions are induced by appending the next bit and shifting the window. In that sense, GPT-style models can be seen as vastly larger learned autoregressive systems over much richer token alphabets and state representations.

Computational perspective. A transformer language model is not itself a Turing machine because it operates with finite parameters and a finite context window. In that sense it behaves like a very large learned state machine. However, if the generated sequence is treated as external memory and the model repeatedly reads and writes to it, the system can emulate arbitrary algorithms. For this reason transformer architectures are often described as computationally universal in principle.

Architecture. The model is a single-block decoder-only transformer. Each input bit xi ∈ {0,1} is mapped to a learned token embedding and added to a learned positional embedding. The resulting length-k sequence is processed by masked multi-head self-attention, where the mask enforces that position i can attend only to positions j ≤ i. This is followed by a position-wise feed-forward MLP, with residual connections and layer normalisation around both sublayers. The hidden vector at the final position is then projected to two logits, which are converted by softmax into P(xt+1=0 | xt-k+1:t) and P(xt+1=1 | xt-k+1:t).

Usage. Choose a preset rule or sequence, set the context window, click Build data, then Train model. Compare the target and learned transition graphs and inspect the predicted next-bit probabilities for each context state.
