2. Impulse response h[n]

First click Measure impulse response h[n]. This applies a unit impulse at n = 0 and records the resulting motion. We will then treat this measured sequence as the system signature.

3. Input force u[n]

Any discrete input can be written as a weighted impulse train: u[n] = Σₖ u[k] δ[n-k]. This means that at each time index k we place a shifted unit impulse δ[n-k] and scale it by the input value u[k]. For the current input, the highlighted simple case is shown here.

4. Output: direct simulation vs convolution

Blue is the true simulated motion. Purple is the response reconstructed from h[n] using convolution. For a single delayed impulse u[n] = δ[n-k₀], linearity and shift invariance give y[n] ≈ h[n-k₀]. For a general weighted impulse train, we add these delayed responses: y[n] ≈ Σₖ u[k] h[n-k].
