GAN Demo: Learning a Coin Toss Process

Set an initial probability of heads, generate a large training sequence of heads and tails, and train a GAN in the browser to mimic that distribution from random noise alone. The demo compares the true and generated probabilities of heads, shows training losses, visualises the learned latent-to-output mapping, and prints example generated sequences.

Notes

For this Bernoulli example, a GAN is deliberately excessive. The point is to show that a generator can start from random noise and learn to imitate an unknown data source just by competing with a discriminator.

Because the data are binary but the generator is differentiable, the model is trained using continuous outputs in [0,1]. For display and sample generation, these outputs are converted into heads and tails by Bernoulli sampling.

The training process is stochastic. Running it again with the same target probability may still produce slightly different results.
