# Data Mining and Machine Learning

This is the repository for our 10 week Data Mining and Machine Learning Module (CS909 and CS429) (previously Data Mining) at the University of Warwick by Fayyaz Minhas.


[Official Website](https://warwick.ac.uk/fac/sci/dcs/teaching/material/cs909/)


[Videos and class questions for each week](https://sites.google.com/view/fayyaz/courses/data-mining?authuser=0)

Various Demos Can be accessed as: https://foxtrotmike.github.io/CS909/<demo name>.html

(e.g., https://foxtrotmike.github.io/CS909/association.html)
